(inner) createProtoMethod(ClassType, messageName, preString)

Creates a method that becomes a part of the protobuf singleton object. It creates two method:
  • The first method is the name of the message. This creates an instance of the message. An example is Request calling PROTOBUF_UTIL.Request() creates an instance of the Request class. If the message is an enum then you can call it without the () an example is: PROTOBUF_UTIL.CommandType.ADD_STROKE
  • The second method is the class of the message. This can be useful to get access to messages inside messages. This second method only exist for messages and does not exist for enums. an example is: PROTOBUF_UTIL.getRequestClass()
Parameters:
Name Type Description
ClassType function | Enum the actual data that represents the protobuf data. If the classType is not a function then we treat it like an enum.
messageName String the name of the message.
preString String a string that is used to preprend the messageName. This can be used to prevent conflicts. The value must not be undefined.
